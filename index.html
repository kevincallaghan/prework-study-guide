HTML

  • The head element contains information about the webpage.
  • Information in the head is not visible on the webpage
  • The body element represents the visible content shown to the user.
  • The body consists of the parent body element and its child elements
  • The body is a semantic HTML element, where the name of the element describes its content and the role it plays on a webpage
  • These child elements in the body give structure to the webpage

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS can be added three ways: inline CSS, internal CSS style sheet, and external CSS style sheet
  • CSS Rule consists of a selector, property, and value: ex - div {color: blue;}. The selector defines the element to which the rules apply and the declarations contain the CSS property and the value for that property.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git Add -A: commit all files in the working branch
  • git commit -m "[descriptive message]": commit your staged content as a new commit snapshot
  • git pull: fetch and merge any commits from the tracking remote branch
  • git push [alias] [branch]: Transmits local branch commits to the remote repository branch

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Types of Variables:
    1. String - sequence of text enclosed in quotes
    2. number - numeric values not enclosed in quotes
    3. Boolean - True or False - Special values that do not need quotes
    4. Array - Structure that allows you to store multiple values in a single reference
    5. Object - Can be anything - Everything in JavaScript is an object and can be stored in a variable
  • Control flow is the order in which a computer executes code in a script.
  • Comments are snippets of text that can be added along with code and ignored by the browser
  • Conditionals are code structures used to test if an expression returns true or not. Commonly performed with if...else statements
  • Events are code structures that listen for activity in the browser and run code in response
  • Array is a single variable used to hold a group of data, most often related.
  • For Loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over
  • Function is a set of instructions that tells the computer how to perform a certain task.
  • Functions do not automatically execute as they must be called by name
